## 剩余风险 / 后续建议

1. 本轮主承重使用组合 + 时间信号；引用信号仅确认数据源存在，未进入主承重。若后续需要“引用空白”，应单独做 M070b 小桩并证明新增信号显著。
2. 概念抽取采用标题 n-gram，确定性强但语义归并有限；正式产品展示前可增加离线同义归并，但必须重跑同一预测性承重。
3. LLM 叙述只做接地解释；任何作者/年份/DOI/期刊等语料未提供字段，仍不得补写，需通过连接器或人工核验。
