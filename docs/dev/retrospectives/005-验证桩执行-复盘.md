### 14.5 有界决策与偏差说明

- 为适配 `deepseek-v4-flash` 推理模型，T2b 将 LLM 展示候选数限制为 50、摘要截断为 120、`max_tokens=8192`，并对空 content / 不可解析 JSON raw 做隔离重试；这些只影响执行稳定性与成本，不改变同候选池、0 虚构、temperature=0、原始响应落盘等判据。
- T5 的「每臂 N=3」按小样本样本数执行，而不是 10 查询 × 3 轮重复调用；理由是 plan 明确本阶段为 spike 小样本，且 temperature=0 / seed=42 下重复调用不提供额外随机方差信息，却会显著增加 token/延时成本。
- T2a embedding 仍是 LSA（TF-IDF+SVD），不是神经稠密检索；已在 `040-验证桩结论.md` 前进路径中列为正式 X2 对比前置。
