## 5. 风险与后续

- OpenAlex 元数据会随时间变化；M050 以缓存快照 + `fetched_at` 做复现基准，在线再取可能更新。
- Crossref 对部分 arXiv DOI 返回失败，已记录为 prior_failures；这不影响主源 OpenAlex 的 C1/C2 判据，但正式产品可把 arXiv API 作为这些 DOI 的补源。
- Semantic Scholar/arXiv 连接器已实现为备用层，本轮未把它们纳入大规模填充路径，避免在 OpenAlex 已覆盖时增加额外限速与许可复杂度。
