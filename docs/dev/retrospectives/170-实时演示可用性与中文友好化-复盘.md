## 1. 本轮做了什么

1. **实时缓存提速**：`run_realtime_query` 跨调用复用语料、BM25、DenseV2、CrossEncoderReranker；新增 `warm_realtime_cache()` 和 `clear_realtime_runtime_cache()`。
2. **预算与诚实成本**：默认 timeout 从 45s 放宽到 180s；已算完的真实结果不再因预算阈值被丢弃；失败路径保留“不编造”，并记录真实耗时。
3. **实时结果卡中文友好化**：实时卡显示「已将问题拆解为」「第 N 名 · 论文 X · 综合得分」「排序依据」等中文标签；英文论文标题/摘要保持原文。
4. **排序技术细节折叠**：原始 `bm25=...*0.10` 权重串不再占主卡，折叠进「查看技术细节」。
5. **点即核验面板中文化**：`manual_review_reason` 等后端状态码在 JS 显示层映射为中文解释；`source_text.py` / `interactive.py` 未改。
