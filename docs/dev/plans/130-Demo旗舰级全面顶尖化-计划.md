## 1. 模块目标

- **本模块解决什么**：把 Demo 从"忠实但局部"升级为**全面旗舰级顶尖体验**——①**实时自主搜索主体验**（切题：当场问任意复杂问题→实时拆解→64k 语料检索→排序→理由）；②**顶尖观感**（设计令牌驱动的统一视觉语言 + 微交互 + 状态设计 + 响应式）；③**关系图稳定**（确定性布局、坐标冻结、零抖动、resize 稳）；并把**点即核验 + 推理轨迹**整合进旗舰统一体验。
- **对冻结成功标准的贡献**：
  - **S4 落地（15%）**：实时自主搜索直接展示 §3.1 核心功能（"智能论文搜索"真能跑），补当前"离线查看器"缺口。
  - **S3 结构化（10%）**：稳定关系图 + 渐进式轨迹 + 证据矩阵的旗舰级结构化呈现。
  - **S5 / H5**：观感升级不破忠实——点即核验仍是 fabrication=0 现场证明；实时结果诚实标注。
- **诚实性要求**：实时结果非 verified 承重，明确标注 + 成本披露 + 失败优雅回退不编造；观感/动效不得引入任何超 verified 证据的展示或假数据。

## 2. 范围

- **本轮必须做（三支柱，均守忠实/稳定/诚实）**：
  1. **T1 · 实时自主搜索主体验（切题核心）**：旗舰页 hero 输入框 → 调 `realtime.run_realtime_query`（已存，A-v2 兼容在线路径，搜 LitSearch 64k）→ 展示**拆解子查询 + 排序结果（标题/摘要/理由/分数）+ 成本（调用/token/延时）**；**实时"检索中"状态**（loading）、**优雅回退**（实时关闭/LLM 不可用/超时→显式提示，不编造）；明确标"实时·非确定性·非 verified 承重"，与基准 verified 路径分离。
  2. **T2 · 顶尖观感/设计系统**：建**设计令牌**（色彩/间距/字体阶梯/圆角/阴影/动效，统一变量）；精炼组件（卡片/徽章/表格/输入/按钮/标签页）；**微交互**（hover/focus/过渡，平滑非抖动）；**状态设计**（loading/empty/error/timeout 全覆盖）；信息层次与留白；**响应式**（桌面/窄屏）；可访问性（语义/对比/键盘）。**轻量内联 CSS/JS、无前端框架**（守 035）。
  3. **T3 · 关系图稳定化（线条稳定·一切问题）**：**确定性布局**——服务端/种子固定算一次、坐标冻结（端侧**不跑力导向模拟**=构造零抖动）；节点碰撞间距（标签不重叠）；边锚定节点中心、稳定路径、语义着色（如未来填补）；**viewBox SVG**（resize 只缩放不重排）；hover/focus 高亮邻居**仅切类不改坐标**（无闪烁）；数据来自 verified JSON（M070），缺失显式空状态。**确定性断言**：同数据两次渲染坐标逐字相等。
  4. **T4 · 整合点即核验 + 轨迹进旗舰**：把 M120 点即核验（复用 `source_text.verify_value_span`，高亮==char_span==value）+ 推理轨迹整合进旗舰统一体验、保持忠实。
  5. **合规 / 可复现 / 复盘**。
- **明确不做**：改 M010–M110 任何产物/源码行为/判据；改 M080 既有面板/`render.py`/`verify.py` 与 M120 `interactive.py`/`source_text.py` 既有逻辑（**新增**旗舰视图/端点、复用其 verified 函数）；引重前端框架（React/Vue 等）；编造证据/实时结果/置信；把实时结果当 verified 承重；放身份信息；为观感牺牲忠实/稳定/性能/可复现。
- **允许修改 / 新建的文件**：
  - 新建：`src/scholarloop/demo/studio.py`（旗舰视图渲染 + 实时搜索 UI + 整合）、`src/scholarloop/demo/graph_layout.py`（确定性稳定布局）、`src/scholarloop/demo/design.py`（设计令牌/CSS/JS 资产，可内联）；demo 路由**新增**端点（如 `/studio`、`/api/search`（实时）、`/api/graph_stable`）。
  - 新建测试：`tests/test_m130_*.py`（**不改** `test_m010..m120`）。
  - 新建产物：`reports/m130/**`（旗舰 fidelity、图确定性校验、实时烟测、secret/identity 扫描、validation_summary）。

## 4. 调研结论

- **来源**：顶尖 Web 设计系统/设计令牌（2026）、D3-force 确定性布局/冻结坐标避抖、AI Agent 信任 UX（上轮）、竞赛获奖呈现要素。
- **可直接落地的结论**：
  1. **切题**：实时自主搜索是 §3.1 核心，必须做成可当场演示的主体验。
  2. **观感**：设计令牌 + 微交互 + 状态设计 = 旗舰级"高级感"，轻栈即可达成。
  3. **稳定**：端侧零力导向模拟 + 冻结坐标 + viewBox = 线条稳定的根本解。
- **对本计划的影响**：承重 = 实时诚实 + 观感不破忠实 + 图确定性稳定 + 不破 M080/M120。
