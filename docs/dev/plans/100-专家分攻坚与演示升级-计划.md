## 1. 模块目标

- **本模块解决什么**：F1 封顶后，把分数劲转到**能赢的专家分 40% + 结构化 10%**——①**硬化创新**（M070 研究空白做频率配平消融，证明"剥离频率效应后空白预测信号仍成立/或诚实标注边界"）；②**满足官方"关系图"**（可视化节点连线图，超越 M080 的矩阵表格）；③**强化落地观感**（实时提问演示模式，系统真跑新查询）；④提交物料同步硬化创新/泛化叙事。
- **对冻结成功标准的贡献**：
  - **S4 创新（15%）**：频率配平消融让"可证伪研究空白"主张**问不倒**（评审最会质疑频率混杂，本模块先把它消掉）。
  - **S3 结构化（10%）**：可视化关系图对齐官方"列表、关系图"明文要求。
  - **S4 落地（15%）**：实时提问模式展示系统**真能跑**复杂新查询（补 M030/M080 的"离线查看器"局限）。
  - **S5 / H5**：消融可复现、关系图值溯源 verified JSON、实时结果不编造（区分证据状态、未解析标"需人工核验"）。
- **诚实性要求**：频率消融若显示空白信号**减弱或消失**，**如实上报并调整创新主张**，绝不藏；实时模式失败/超时**优雅回退 + 明确标注**，绝不编造结果。

## 2. 范围

- **本轮必须做**：
  1. **T1 · M070 频率配平消融（创新硬化）**：构造**频率匹配的随机基线**（按概念边际频率配平，剥离"候选是高活跃概念对"的混杂），重算空白预测效力是否仍显著 > 该配平基线；确定性、bootstrap/置换。产出 `reports/m100/gap_frequency_ablation.json` + 结论（成立则创新更硬，减弱则诚实标注边界）。**不改 M070 verified 产物。**
  2. **T2 · 可视化关系图（结构化）**：研究空白/证据的**节点连线图**（概念为节点、共现/引用/未来填补为边），纯前端轻量 SVG/Canvas（不引重前端框架），数据**逐字来自 verified JSON**（M070 gaps_display / M020 evidence）、零伪造、缺失不补写。扩展 M080 demo（**新增**视图，不改其 verified 面板与 fidelity）。

## 4. 调研结论

- **来源**：官方 §3.1.4「关系图」+ §4 专家分；M070/M080 携带项；M090 F1 封顶结论；015 S3/S4/S5。
- **可直接落地的结论**：
  1. 专家分是当前**最高赢面**：创新（可证伪空白）、落地（真连接器+实时）、泛化（跨基准）均为强项，打磨即得分。
  2. 频率配平消融是**把创新主张从"可被质疑"变"问不倒"**的关键一招。
  3. 关系图 + 实时模式补齐官方明文要求与"真能跑"观感，且可在零伪造/成本透明下做。
- **对本计划的影响**：承重转为「消融诚实可复现 + 关系图忠实零伪造 + 实时成本透明 + 上游冻结」。
