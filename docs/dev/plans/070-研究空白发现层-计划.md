## 1. 模块目标

- **本模块解决什么**：补 S4 **创新（15%）** 的核心差异化件——研究空白发现。把 A 流水线（检索排序）+ B-lite（证据矩阵）的下游延伸为：从语料结构中**确定性地检出"研究空白候选"**，再由 LLM **只对真实论文做接地叙述**，形成"可证伪、证据接地、零虚构"的空白发现能力。
- **对冻结成功标准的贡献**：
  - **S4 创新（15%）**：本模块主交付——研究空白发现，且以**可证伪的预测性信号**支撑，区别于"LLM 拍脑袋说空白"。
  - **X1（不自标注）**：空白用**结构化/操作化定义**（共现/引用/时间信号），不靠专家或 LLM 自标注"这是有价值的空白"。
  - **H5（0 虚构）/ S5（合规）**：每条空白候选必须**可溯源到语料事实**（哪些论文存在/不存在、共现计数），LLM 叙述不得新增论文/DOI/作者/年份；区分"已有证据支持/证据不足/存在争议"。
  - **X2/可复现**：同语料、确定性、temperature=0/seed，继承 M040/M060。
- **诚实性要求**：若结构化空白**无预测效力**（schema 桩不过），如实停机上报"研究空白发现不具可证伪信号"，**绝不**用 LLM 叙述冒充发现、绝不自标注价值。

## 2. 范围

- **本轮必须做**：
  1. **T0 · schema 桩（gate，make-or-break）**：小样本证明"结构化空白候选 + 预测性验证"可成立——定义空白输出 schema（每条带证据溯源），在**时间切片**上检测、验证其**预测效力显著 > 随机基线**。桩不过即停机。
  2. **结构化空白检测器**（确定性）：从语料三类信号检出空白候选——①**组合空白**（子主题 A、B 各自高频但近乎不共现）；②**引用空白**（高被引论文邻域覆盖稀疏，用 LitSearch `citations` 列）；③**时间空白**（曾活跃后沉寂的主题，用 arxiv 号年月 / RealScholarQuery `published_time`）。
  3. **接地叙述层**：LLM 仅对检出候选所**关联的真实论文**做自然语言说明，**不新增任何事实**；输出挂证据溯源（论文 id、共现/引用/时间计数）。
  4. **预测性承重评测**：在"过去切片"检测空白，验证被"未来留出切片"真实论文填补的比例**显著高于随机概念对基线**（bootstrap≥10000/置换，CI 不含 0 或 p<0.05）。
  5. **结构化展示对接**：空白结果以**列表 + 关系/矩阵**结构呈现（承接 S3/B-lite 形态，不做大交互图谱）。
  6. **合规 / 可复现 / 复盘**。
- **明确不做**：改 M010–M060 产物/判据/A-v2 配置（全冻结只读）；**自标注空白价值**（X1 红线）；让 LLM 新增论文/事实（H5 红线）；大规模交互知识图谱（035 有界约束）；重训模型；付费/受限数据；任何虚构。
- **允许修改 / 新建的文件**：
  - 新建：`src/scholarloop/gaps/**`（检测器 + 接地叙述 + schema）、评测入口 `src/scholarloop/eval/run_gaps.py`、schema 桩 `spike/gaps/run_gap_spike.py`。
  - 新建测试：`tests/test_m070_*.py`（**不改** `test_m010..m060`）。
  - 新建产物：`reports/m070/**`（含 `spike/`、`cache/`、`raw/`）。
  - **不得**改动：FROZEN 件、既有 `spike/**`（新建在 `spike/gaps/`）、`reports/m010..m060/**`、M010–M060 既有源码行为、各模块验收判据。

## 3. 现状与证据

- **已存在可复用**：RealScholarQuery 语料（`reports/m060/cache/realscholarquery/corpus.jsonl`，arxiv 号含年月）+ 50 查询官方 gold（带 `published_time`）；LitSearch 语料（含 `citations` 列）；M020 证据矩阵接地范式；M040/M060 的 bootstrap/置换、同候选池、H5、确定性协议。
- **信号源已核实**：RealScholarQuery gold `source_meta.published_time`（如 "20241001"）；arxiv 号 YYMM 覆盖 0704–2410（可时间切片）；LitSearch `citations`。
- **仍未知但 T0 桩须先回答**：结构化空白是否具**预测效力**（最大风险，§11）；概念抽取的确定性口径（n-gram/受控词表，避免 LLM 主观）；预测信号的样本量是否足够显著。

## 4. 调研结论

- **来源**：035 第 4 步（条件触发 + schema 桩前置）、015 S4（创新点名研究空白发现）、02 评审 X1/X2、M060 verified 语料/时间信号、文献中"未探索组合 / citation gap / dormant topic"的可操作定义。
- **可直接落地的结论**：
  1. 研究空白可**操作化为结构信号**（共现/引用/时间），**确定性可算、可证伪**，避开 X1 自标注。
  2. **预测性验证**（过去检出→未来填补率 > 随机）是把"空白发现"变成**可证伪承重墙**的关键，而非靠人觉得"有道理"。
  3. LLM 只做**接地叙述**（解释已检出候选关联的真实论文），承重不依赖 LLM 主张，H5 守得住。
- **对本计划的影响**：T0 schema 桩（预测效力）是 make-or-break；不过则停机，**不得**降级为"LLM 叙述即发现"。

## 5. skills 选型与执行简报

- **选择的 skills**：无（检测器 + 预测性评测为确定性工程任务；接地叙述复用 M020 范式）。
- **未选其它 skills 的理由**：承重在**可证伪结构信号**，靠协议而非编排保证；LLM 仅受限叙述。
- **转译后的普通执行步骤（硬要求标【硬】 / 建议标【议】）**：
  1. 【硬】空白用**操作化结构定义**，不自标注价值（X1）；每条候选挂语料证据溯源。
  2. 【硬】**T0 schema 桩**先过：预测效力显著 > 随机基线，否则停机。
  3. 【硬】LLM **只接地叙述真实论文，不新增事实**（H5）；区分证据状态（S5）。
  4. 【硬】同语料、temperature=0/seed、bootstrap/置换、原始落盘继承 M040/M060。
  5. 【硬】上游 M010–M060 冻结只读；新产物写 `reports/m070/`、`spike/gaps/`。
  6. 【议】概念抽取实现（n-gram/受控词表/KeyBERT 离线）、切片边界、缓存键，在 §10 预算内定并落盘依据。

## 6. 任务拆解

- [ ] **T0（不确定度：高·make-or-break）schema 桩**：小样本定义空白 schema（带证据溯源）+ 在时间切片上检测组合空白 + 验证预测效力（过去检出被未来填补率 vs 随机概念对基线）显著。产出 `reports/m070/spike/gap_schema_spike.json` + 结论。**【停机】** 预测效力不显著 / 需自标注价值 / 概念抽取无法确定性化 → 上报，不建主体。
- [ ] **T1（不确定度：中）结构化检测器**：`gaps/detect.py` 实现组合/引用/时间三类空白，确定性、缓存；单测覆盖检测可复现与证据溯源完整。
- [ ] **T2（不确定度：中）接地叙述层**：`gaps/narrate.py`，LLM 仅对候选关联真实论文叙述，挂溯源；fabrication 校验（叙述提及的论文 id 必在候选证据集内，0 越界）。
- [ ] **T3（不确定度：高）预测性承重评测** ← make-or-break：`eval/run_gaps.py` 全量跑预测效力 + bootstrap/置换 + 随机基线；产出 `reports/m070/results.json`+`significance.json`+`评测报告.md`。
- [ ] **T4（不确定度：低）结构化展示对接**：空白结果导出为列表 + 关系/矩阵结构（B-lite 形态），落 `reports/m070/gaps_display.json`。
- [ ] **T5（不确定度：低）合规 / 可复现 / 复盘**：无密钥；缓存重放一致；复盘含"空白发现成立/不成立"诚实结论。

## 11. 风险、边界与回滚

- **核心风险（T0/T3）**：结构化空白可能**无预测效力**（检出的"空白"只是噪声）。**这是真实信号** → 停机上报，由人类决定（接受"研究空白发现不成立"的诚实结论 / 换信号 / 降级为纯结构化统计展示而不称"发现"），**绝不**用 LLM 叙述冒充发现或自标注价值。
- **X1 红线**：空白只用操作化结构定义 + 预测性验证，不自标注价值。
- **H5 红线**：LLM 只接地叙述真实论文，零新增事实。
- **有界约束**：只做列表/矩阵结构化展示，**不**做大规模交互知识图谱（035）。
- **回滚**：本模块为新增层，上游全部冻结只读；失败仅废弃 `src/scholarloop/gaps/**`、`spike/gaps/**` 与 `reports/m070/**`，不污染已 verified 证据。
