## 1. 模块目标

- **本模块解决什么**：把提交物料的创新叙事与 M100 频率消融**对齐**——消除每一处被证伪的"预测性研究空白"主张，重定位创新主轴，使整套提交物料**内部一致、零假主张、可辩护**。
- **对冻结成功标准的贡献**：
  - **S4 创新（15%）**：从"一个可能被推翻的强主张"换成"一组扛得住质疑的真主张 + 自我证伪方法论"，**降低被评审证伪的风险、提升可信度**。
  - **S5 合规 / 零夸大**：提交物料不携带已撤回主张，每条主张挂仍成立的 verified 证据。
- **诚实性要求**：研究空白**诚实降级**为"候选方向生成启发"（确定性、零虚构、零自标注仍成立），**不再claim 预测有效性**；并把频率消融作为**严谨性正面证据**呈现（敢自证）。

## 4. 调研结论

- **来源**：M100 第十六次初查（频率消融证伪 + 战略裁定）、015 S4/S5、09-M100 口径。
- **可直接落地的结论**：
  1. 携带已撤回主张是**最大合规/可信风险**——必须清零。
  2. 真实差异化（忠实证据链/真出处/跨基准泛化/自我证伪）足以支撑一个**扛得住质疑**的创新叙事。

- **对本计划的影响**：承重 = 提交物料零残留假主张 + 创新叙事可辩护 + 内部一致。
