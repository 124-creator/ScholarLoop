## 提交版脱敏说明

M050 开发记录中的 polite UA 含联系邮箱；提交版统一写作 `mailto:<redacted>`，不展示真实联系信息。来源：`reports/m050/data-sources.md`，脱敏核验：`reports/m080/submission_identity_scan.json` 与 `reports/m080/secret_scan.json`。

## 不做的事

- 不爬全文。
- 不绕过 API 限速或许可要求。
- 不把密钥、token 或真实联系邮箱写入提交材料。
- 不把在线实时变化数据作为本次提交指标来源；评测与 Demo 均使用 verified 快照。
