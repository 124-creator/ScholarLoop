# 创新叙事（可溯源版）

## 1. 定位

PaSa、SPAR、Ai2 Paper Finder 等系统强调多智能体检索、查询演化或多索引采样。ScholarLoop 的差异化定位是：在公开基准排序之外，把推荐结果做成可复核证据链、真实出处快照和跨基准泛化证据，并把不成立的强主张主动用消融桩推翻和收紧，而不是让 LLM 主观判断“哪里有空白”。赛题参考来源：`00-赛题/科研场景下复杂学术查询的智能论文搜索与推荐.txt`。

## 2. 已验证创新点

1. **忠实零伪造证据链**：M020 对每条卡片记录 source_field、char_span 与证据状态；M080 逐字渲染，不补写缺失字段。M020 verification 中 `fabrication_rate=0.0`、`matrix_fabrication_rate=0.0`，M080 fidelity 中 fabrication=`0`、out_of_pool=`0`。来源：`reports/m020/verification.json`、`reports/m020/evidence/*.json`、`reports/m080/fidelity_audit.json`。
2. **真实元数据连接器**：M050 用 OpenAlex / Crossref 等开放元数据快照补作者、年份、DOI 与出处；未解析字段保留人工核验提示，不把空缺编成事实。提交文案只写脱敏 UA，不暴露联系信息。来源：`reports/m050/enriched_replay/*.json`、`reports/m050/data-sources.md`。
3. **跨基准零样本泛化**：A-v2 在 LitSearch F1=`0.1311870491`，RealScholarQuery resolvable F1=`0.1972078900`；RealScholarQuery 相对 BM25 的 ΔF1=`0.0913637814`，CI95 `[0.0657101984, 0.1176225304]`，p≈`0.0001`。来源：`reports/m040/results.json`、`reports/m060/results.json`、`reports/m060/significance.json`。
4. **自我证伪方法论**：M070 的研究空白结果保留为“高活跃、零历史共现”候选生成启发；M100 频率配平消融正面检验并推翻旧强主张：候选填补率 `0.108` 低于频率匹配随机对 `0.306`，频率配平 Δ=`-0.198`。因此提交材料不声称频率无关的独立预测能力，而把这件事作为“发现风险—建桩检验—诚实收紧”的严谨性证据。来源：`reports/m070/significance.json`、`reports/m100/gap_frequency_ablation.json`、`docs/dev/000-决策日志.md`。

## 3. 诚实边界

- 研究空白只作为候选方向生成启发，不作为频率无关的独立预测能力；M100 的负结果必须保留并正面呈现。
- 不把 M040 cross-encoder 的小幅 F1 边际夸大为主要贡献；其更强证据在 NDCG@20 边际。
- M060 第二基准未把 cross-encoder 单独作为消融承重；提交主张只说 A-v2 冻结配置整体迁移显著优于 BM25。
- Demo 不新增 F1 指标，不现场联网，不调用 LLM；实时模式如启用也与 verified 离线路径分离。
