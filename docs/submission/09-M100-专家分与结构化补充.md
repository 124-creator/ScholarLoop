# M100 专家分与结构化补充

本页为追加材料，只补充 M100 新增证据，不改前序 verified 主张。

## 1. 创新边界：频率配平消融后的诚实结论

- 证据：`reports/m100/gap_frequency_ablation.json`
- 原 M070 随机基线：候选未来填补率 0.108，对照 0.022，差值 0.086，95% CI [0.056, 0.116]。
- M100 频率配平后：候选未来填补率 0.108，频率匹配对照 0.306，差值 -0.198，95% CI [-0.246, -0.150]。
- 结论：控制概念边际频率后，M070 的空白预测信号减弱或消失。创新主张收紧为：系统能给出“高活跃、零历史共现”的候选生成启发，但不声称已证明频率无关的独立预测能力。

- 入口：`/api/realtime?q=...`
