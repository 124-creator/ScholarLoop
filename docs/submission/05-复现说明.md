# 复现说明

## 1. 环境

- Python 包路径：`src`
- Demo 默认：offline、0 LLM 调用、只读 verified JSON。
- 真实元数据演示读 `reports/m050/enriched_replay/`，不重新联网。

## 2. 单命令启动 Demo

```powershell
$env:PYTHONPATH="src"
python -m scholarloop.demo.app --host 127.0.0.1 --port 8766
```

访问：`http://127.0.0.1:8766`

## 3. 忠实校验
