# T4 · S-A2 证据归因准确率定义

## 指标定义
- 判定单位：每篇论文 · 每个证据字段实例。本次试算只统计方法 / 数据 / 结论三个核心字段，共 10 篇 × 3 = 30 个实例。
- 正确定义：字段内容必须由同一论文的元数据、摘要或开放片段直接支持；不得外推、不得补写来源没有的信息。
- 错误定义： DOI/作者/年份不匹配、字段内容无出处、把相邻论文信息嫁接到当前论文、或用 LLM 常识补全。
- 利益冲突缓解：执行方先判并保留逐字段出处；Claude Code 初查或人类终验抽样复判至少 30% 字段。若复判与执行方不一致，以复判为准并回写差异。

## 试算
- 样本量：30 个字段实例。
- 有明确出处：28 个。
- 归因准确率试算值：93.3%。
- 定义完整性判定：PASS
